Instruções

Use 'CRTL+SHIFT+C' para inserir codigos a pagina

Use 'CRTL+SHIFT+M' para marcar texto

Use 'CRTL+SHIFT+X' para marcar texto com código

Click no '+' à esquerda para selecionar tipo do bloco(da linha)

Click nos '. . .' à direita para mover, exluir ou realizar outras opções de linha

Selecione um trecho de linha para visualizar as caixas de opção
